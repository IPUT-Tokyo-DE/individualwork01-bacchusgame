# 避けるボーイ

## ゲームの内容
ここにゲームの内容説明を記述する
”避ける”をテーマにしたアクションゲーム
レーザー、爆弾、のこぎり、迫りくる攻撃をよけできるだけ長く生き残れ！
あたってしまうと体分離してしまう。臨機応援に行動しよう！
